# Phase 2 — Authentication Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add email + password authentication to the backend: register, login, logout, and a protected "current user" endpoint, using argon2 password hashing and opaque Redis sessions delivered via an httpOnly cookie.

**Architecture:** Password hashing lives in `app/security.py` (argon2-cffi). Sessions live in `app/sessions.py` (random opaque token → user id in Redis with a TTL). Request/response shapes are Pydantic schemas in `app/schemas.py`. The auth endpoints are an `/auth` router (`app/routers/auth.py`); a reusable `get_current_user` dependency (`app/deps.py`) reads the session cookie and loads the user. No new database tables — the `users` table already exists from Phase 1, so there is no migration.

**Tech Stack:** Python 3.12, FastAPI, SQLAlchemy 2.0, Redis, argon2-cffi, email-validator (for `EmailStr`), pytest, httpx TestClient. Postgres + Redis run via the Phase 1 Docker Compose stack.

## Global Constraints

- Run all commands from `backend/` with the venv activated (`. .venv/bin/activate`). Postgres + Redis must be up (`docker compose up -d postgres redis`).
- Password hashing uses **argon2** (`argon2-cffi`).
- Sessions: a random opaque token (`secrets.token_urlsafe(32)`) → user id, stored in Redis under key `session:<token>` with TTL `settings.session_ttl_seconds`. Delivered as an **httpOnly**, `SameSite=Lax` cookie named `settings.session_cookie_name` (default `session`), `secure=settings.cookie_secure` (default `False` for local dev).
- Endpoints and status codes (exact):
  - `POST /auth/register` → `201` with `{id, email}`; duplicate email → `409`; invalid body → `422`.
  - `POST /auth/login` → `200` with `{id, email}` and sets the session cookie; wrong email/password → `401`.
  - `POST /auth/logout` → `204`, deletes the session from Redis and clears the cookie.
  - `GET /auth/me` → `200` with `{id, email}` when authenticated; missing/invalid/expired session → `401`.
- Password minimum length: **8** characters (enforced by the register schema).
- SQLAlchemy 2.0 style; DB driver `postgresql+psycopg://`.
- Register does NOT auto-login (no session issued); the client logs in separately. Login is the only endpoint that issues a session.
- Every task ends with a passing test run and a commit.

```python
from argon2 import PasswordHasher
from argon2.exceptions import InvalidHashError, VerificationError, VerifyMismatchError

_hasher = PasswordHasher()


def hash_password(password: str) -> str:
    return _hasher.hash(password)


def verify_password(password: str, hashed: str) -> bool:
    try:
        return _hasher.verify(hashed, password)
    except (VerifyMismatchError, VerificationError, InvalidHashError):
        return False
```

**Interfaces:**

  - `app.sessions.get_session_user_id(token: str) -> int | None` — returns the user id, or `None` if missing/expired/malformed.

- [ ] **Step 1: Create `app/sessions.py`**

```python
import secrets

from app.config import settings
from app.redis_client import get_redis

_PREFIX = "session:"


def create_session(user_id: int) -> str:
    token = secrets.token_urlsafe(32)
    get_redis().set(_PREFIX + token, str(user_id), ex=settings.session_ttl_seconds)
    return token


def get_session_user_id(token: str) -> int | None:
    value = get_redis().get(_PREFIX + token)
    if value is None:
        return None
    try:
        return int(value)
    except (TypeError, ValueError):
        return None


def delete_session(token: str) -> None:
    get_redis().delete(_PREFIX + token)
```

- [ ] **Step 2: Create `app/routers/auth.py`**

```python
from fastapi import APIRouter, Depends, HTTPException, Request, Response, status
from sqlalchemy import select
from sqlalchemy.orm import Session

from app.config import settings
from app.db import get_session
from app.models import User
from app.schemas import LoginRequest, RegisterRequest, UserOut
from app.security import hash_password, verify_password
from app.sessions import create_session, delete_session

router = APIRouter(prefix="/auth", tags=["auth"])


def _set_session_cookie(response: Response, token: str) -> None:
    response.set_cookie(
        key=settings.session_cookie_name,
        value=token,
        max_age=settings.session_ttl_seconds,
        httponly=True,
        samesite="lax",
        secure=settings.cookie_secure,
        path="/",
    )


@router.post("/register", status_code=status.HTTP_201_CREATED, response_model=UserOut)
def register(payload: RegisterRequest, db: Session = Depends(get_session)) -> User:
    existing = db.scalar(select(User).where(User.email == payload.email))
    if existing is not None:
        raise HTTPException(
            status_code=status.HTTP_409_CONFLICT, detail="Email already registered"
        )
    user = User(email=payload.email, password_hash=hash_password(payload.password))
    db.add(user)
    db.commit()
    db.refresh(user)
    return user


@router.post("/login", response_model=UserOut)
def login(
    payload: LoginRequest, response: Response, db: Session = Depends(get_session)
) -> User:
    user = db.scalar(select(User).where(User.email == payload.email))
    if user is None or not verify_password(payload.password, user.password_hash):
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="Invalid email or password"
        )
    token = create_session(user.id)
    _set_session_cookie(response, token)
    return user


@router.post("/logout", status_code=status.HTTP_204_NO_CONTENT)
def logout(request: Request, response: Response) -> None:
    token = request.cookies.get(settings.session_cookie_name)
    if token:
        delete_session(token)
    response.delete_cookie(settings.session_cookie_name, path="/")
    return None
```

```python
import pytest
from fastapi.testclient import TestClient
from sqlalchemy import create_engine
from sqlalchemy.orm import Session, sessionmaker

import app.models  # noqa: F401  (register mappers on Base.metadata)
from app.config import settings
from app.db import Base, get_session
from app.main import app


@pytest.fixture(scope="session")
def engine():
    eng = create_engine(settings.test_database_url)
    Base.metadata.drop_all(eng)
    Base.metadata.create_all(eng)
    yield eng
    Base.metadata.drop_all(eng)
    eng.dispose()


@pytest.fixture()
def session(engine) -> Session:
    connection = engine.connect()
    transaction = connection.begin()
    factory = sessionmaker(
        bind=connection,
        expire_on_commit=False,
        join_transaction_mode="create_savepoint",
    )
    db = factory()
    try:
        yield db
    finally:
        db.close()
        transaction.rollback()
        connection.close()


@pytest.fixture()
def client(session) -> TestClient:
    def _override_get_session():
        yield session

    app.dependency_overrides[get_session] = _override_get_session
    with TestClient(app) as test_client:
        yield test_client
    app.dependency_overrides.clear()
```

Note: `join_transaction_mode="create_savepoint"` makes each endpoint `db.commit()` operate on a SAVEPOINT that the outer `transaction.rollback()` discards — so endpoint writes are isolated per test. The existing Phase 1 model tests (which only `flush()`) keep working unchanged.

- Produces:
  - `app.deps.get_current_user(request: Request, db: Session = Depends(get_session)) -> User` — raises `401` if the cookie is missing, the session is unknown/expired, or the user no longer exists.
  - `GET /auth/me` on the auth router → `UserOut` for the authenticated user.

- [ ] **Step 1: Create `app/deps.py`**

```python
from fastapi import Depends, HTTPException, Request, status
from sqlalchemy.orm import Session

from app.config import settings
from app.db import get_session
from app.models import User
from app.sessions import get_session_user_id


def get_current_user(
    request: Request, db: Session = Depends(get_session)
) -> User:
    token = request.cookies.get(settings.session_cookie_name)
    if not token:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="Not authenticated"
        )
    user_id = get_session_user_id(token)
    if user_id is None:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="Invalid or expired session"
        )
    user = db.get(User, user_id)
    if user is None:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED, detail="User no longer exists"
        )
    return user
```
